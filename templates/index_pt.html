Sobre mim

Olá! Meu nome é Vinicius Enari e sou formado pela Minerva University, onde obtive meu Bacharelado em Ciência da Computação. Além da minha paixão por programação, também gosto de jogar videogames, ouvir música e tocar saxofone e guitarra elétrica.

Como programador, tenho experiência em construção e utilização de APIs e em construção de aplicativos web com Flask, Django e React. Tenho habilidades sólidas em Python, além de um bom entendimento de JavaScript, C#, PHP e Rust. Também estou aberto a aprender qualquer nova linguagem ou tecnologia se necessário para um projeto ou tarefa.

Sempre estou procurando novos desafios e oportunidades de aprendizado e crescimento profissional. Estou animado em aplicar minhas habilidades e experiência para fazer uma contribuição significativa a qualquer equipe ou projeto em que esteja envolvido. Obrigado por dedicar um tempo para visitar meu portfólio e conhecer mais sobre mim.

Projetos

Twitch Highlights Bot

Este bot em Python usa a API do Twitch para coletar os clips mais vistos da semana anterior de um jogo específico. Em seguida, combina esses clips em um único vídeo compilado usando a biblioteca Moviepy e faz upload do produto final para o YouTube usando a API do YouTube.

Python APIs Google Cloud
Video Directory API

API REST desenvolvida usando o Django Rest Framework para indexar e obter informações de vídeos. Os dados são armazenados no MongoDB Atlas e armazenados em cache usando o Redis para melhorar o desempenho. A API permite que os usuários pesquisem e acessem informações sobre vídeos, incluindo seus URLs, metadados e categorias.

Python Django Rest MongoDB Redis
Twitter Weather Bot

Bot do Twitter que publica previsões do tempo diárias utilizando a API do Twitter e a API do Open Weather. O bot se conecta à API do Open Weather para obters os dados meteorológicos, formata e publica as previsões no Twitter usando a API do Twitter. Ele foi deployado no Heroku e utiliza um scheduler para publicar diariamente.

Python APIs
Twitter Profile Updater

Bot em Python que utiliza a API da NASA Astronomy Picture of the Day para atualizar um perfil do Twitter com imagens impressionantes do espaço. O bot é executado automaticamente diariamente graças ao seu deployamento no Heroku.

Python APIs
Slideshow Moviemaker

Programa em Python que utiliza a biblioteca moviepy para criar vídeos de slideshow a partir de uma coleção de imagens e uma trilha de áudio.

Discourseium

Aplicativo de chat web inspirado no Discord usando React. O aplicativo permite que os usuários se comuniquem entre si em tempo real através de texto. Ele também inclui recursos como o usuário poder usar seus perfis do github e emojis.

JavaScript React NextJS Vercel Supabase
Battlesim

Protótipo de um jogo de estratégia por turnos desenvolvido usando Unity. O jogo é 2D Top-down de um campo de batalha onde o jogador precisa selecionar unidades e colocá-las no campo de batalha para lutar contra o inimigo. Funcionalidades implementadas incluem movimentação de unidades usando o algoritmo de busca A*, capacidade de posicionar unidades arrastando e soltando, e gerenciamento de inventário de unidades.

C# Unity
